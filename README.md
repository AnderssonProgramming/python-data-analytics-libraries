## 🇪🇸 Español

**Propósito:**  
Este repositorio ofrece ejemplos prácticos de las librerías Tier 1 de Python para análisis de datos.  
Cada módulo contiene un script que demuestra flujos de trabajo comunes: carga de datos, estadística descriptiva, visualización y modelado básico.

### 🚀 Estructura

```
python-data-analytics-libraries/
│
├── data/
│   └── sample_dataset.csv
│
├── utils/
│   └── loader.py
│
├── pandas/
│   └── pandas_analysis.py
│
├── numpy/
│   └── numpy_operations.py
│
├── matplotlib/
│   └── matplotlib_visuals.py
│
├── seaborn/
│   └── seaborn_visuals.py
│
├── plotly/
│   └── plotly_interactive.py
│
├── scikit-learn/
│   └── sklearn_model.py
│
├── README.md
└── requirements.txt
```

### 📦 Dependencias

Instala todas las librerías con:

```bash
pip install -r requirements.txt
```

### 📊 Cómo ejecutar cada módulo

1. Clona el repositorio:

```bash
git clone https://github.com/tu_usuario/python-data-analytics-libraries.git
cd python-data-analytics-libraries
```

2. Instala dependencias:

```bash
pip install -r requirements.txt
```

3. Descarga el dataset `data/sample_dataset.csv`.

4. Para cada módulo, navega a su carpeta y ejecuta:

```bash
# Pandas - Análisis tabular y manipulación de datos
cd pandas
python pandas_analysis.py

# NumPy - Operaciones numéricas y matemáticas
cd ../numpy
python numpy_operations.py

# Matplotlib - Visualizaciones estáticas
cd ../matplotlib
python matplotlib_visuals.py

# Seaborn - Visualizaciones estadísticas avanzadas
cd ../seaborn
python seaborn_visuals.py

# Plotly - Gráficos interactivos
cd ../plotly
python plotly_interactive.py

# scikit-learn - Modelado predictivo
cd ../scikit-learn
python sklearn_model.py
```

5. Inspecciona los archivos generados (CSVs, PNGs, HTMLs) en sus respectivos directorios.

### 📊 Outputs Generados

- **Pandas**: Archivos CSV con datos transformados y resúmenes estadísticos
- **NumPy**: Transformaciones numéricas y cálculos matriciales
- **Matplotlib**: Gráficos estáticos de barras, líneas y sectores (.png)
- **Seaborn**: Visualizaciones estadísticas avanzadas como histogramas KDE, boxplots y heatmaps
- **Plotly**: Gráficos interactivos HTML para exploración de datos
- **scikit-learn**: Modelos predictivos y visualización de resultados

### 🔍 Ejemplos destacados

- **Pandas**: Agregación de datos por región y análisis de series temporales
- **Seaborn**: Heatmaps de correlación entre variables numéricas
- **Plotly**: Gráficos interactivos con funcionalidades hover y zoom
- **scikit-learn**: Modelos de regresión para predecir ingresos

## 🇬🇧 English

**Purpose:**  
This repository provides practical examples of Python's Tier 1 libraries for data analysis.  
Each module contains a script that demonstrates common workflows: data loading, descriptive statistics, visualization, and basic modeling.

### 🚀 Structure

```
python-data-analytics-libraries/
│
├── data/
│   └── sample_dataset.csv
│
├── utils/
│   └── loader.py
│
├── pandas/
│   └── pandas_analysis.py
│
├── numpy/
│   └── numpy_operations.py
│
├── matplotlib/
│   └── matplotlib_visuals.py
│
├── seaborn/
│   └── seaborn_visuals.py
│
├── plotly/
│   └── plotly_interactive.py
│
├── scikit-learn/
│   └── sklearn_model.py
│
├── README.md
└── requirements.txt
```

### 📦 Dependencies

Install all libraries with:

```bash
pip install -r requirements.txt
```

### 📊 How to Run Each Module

1. Clone the repository:

```bash
git clone https://github.com/your_username/python-data-analytics-libraries.git
cd python-data-analytics-libraries
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Download the `data/sample_dataset.csv` dataset.

4. For each module, navigate to its folder and execute:

```bash
# Pandas - Tabular analysis and data manipulation
cd pandas
python pandas_analysis.py

# NumPy - Numerical and mathematical operations
cd ../numpy
python numpy_operations.py

# Matplotlib - Static visualizations
cd ../matplotlib
python matplotlib_visuals.py

# Seaborn - Advanced statistical visualizations
cd ../seaborn
python seaborn_visuals.py

# Plotly - Interactive charts
cd ../plotly
python plotly_interactive.py

# scikit-learn - Predictive modeling
cd ../scikit-learn
python sklearn_model.py
```

5. Inspect the generated files (CSVs, PNGs, HTMLs) in their respective directories.

### 📊 Generated Outputs

- **Pandas**: CSV files with transformed data and statistical summaries
- **NumPy**: Numerical transformations and matrix calculations
- **Matplotlib**: Static bar, line, and pie charts (.png)
- **Seaborn**: Advanced statistical visualizations like KDE histograms, boxplots, and heatmaps
- **Plotly**: Interactive HTML charts for data exploration
- **scikit-learn**: Predictive models and results visualization

### 🔍 Highlighted Examples

- **Pandas**: Data aggregation by region and time series analysis
- **Seaborn**: Correlation heatmaps between numerical variables
- **Plotly**: Interactive charts with hover and zoom capabilities
- **scikit-learn**: Regression models to predict revenue
